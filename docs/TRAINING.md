## 2. Training

For multi-GPU training:

```bash
cd RADAR_train
torchrun --nproc_per_node=8 train.py
```

> **Note:** We trained the model using 24 GPUs with a total batch size of 48.

### Configure RADAR+ Global Alignment

Set `radar_plus` to `False` in `radar_config.yaml` to disable RADAR+'s global alignment module. The default value is `True`.

### Train from Scratch vs. Fine-tuning from RADAR Checkpoint

Set `radar_ft` to `True` in `radar_config.yaml` to enable fine-tuning from the pretrained RADAR checkpoint. The default value is `False` (training from scratch).

## 3. Inference

We use the official prompts released by MERLIN during inference. Before running inference, ensure that the MERLIN data has been downloaded, and update the data path:

```python
Modify the img_dir in infer_merlin_anatomy.py and infer_merlin_whole.py
img_dir = "/your_path/merlin_data"
```

We also provide a trained checkpoint (`checkpoint_12.pth`), which is one of our reproduced experiments trained from scratch.

### Anatomy-Level Findings

Single-GPU:

```bash
cd RADAR_train
python infer_merlin_anatomy.py
```

Multi-GPU (recommended):

```bash
cd RADAR_train
torchrun --nproc_per_node=8 infer_merlin_anatomy.py
```

### Whole-Image Findings

Single-GPU:

```bash
cd RADAR_train
python infer_merlin_whole.py
```

Multi-GPU (recommended):

```bash
cd RADAR_train
torchrun --nproc_per_node=8 infer_merlin_whole.py
```

The inference results will be saved as CSV files. We provide two example result files: `checkpoint_12_ZeroShotResult_anatomy.csv` and `checkpoint_12_ZeroShotResult_whole.csv`.
